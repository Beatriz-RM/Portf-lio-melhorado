Projeto Buffet

O Projeto Buffet foi desenvolvido com o objetivo de simular o site institucional de um buffet de eventos. Nele, trabalhei a estruturação com HTML, a estilização com CSS e a organização do layout com foco em responsividade e boa experiência visual. Esse projeto reforçou meus conhecimentos em construção de páginas web, organização de seções e aplicação de boas práticas de design.

Lista de Compras

Esse projeto consiste em uma aplicação funcional para criação de uma lista de compras. Desenvolvi utilizando HTML, CSS e JavaScript, onde o usuário pode adicionar e remover itens dinamicamente. Foi um grande passo para meu aprendizado em lógica de programação e manipulação do DOM, me permitindo colocar em prática interações em tempo real dentro da página.

Seguradora Automotiva

O Projeto Seguradora Automotiva foi criado com a proposta de simular o site de uma empresa de seguros. O objetivo era apresentar serviços, simular orçamentos e transmitir confiança visual ao usuário. Neste projeto, trabalhei a organização de seções com clareza, a paleta de cores institucional e o uso eficiente de imagens e botões, tudo focado na construção de uma interface atrativa e funcional.
